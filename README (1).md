## Adding More Opportunities

To add new residencies/grants:

### For CSV:
1. Open the file in your spreadsheet application
2. Add a new row with all required information
3. Save and share

### For HTML:
1. Open the HTML file in a text editor
2. Find the `data` array in the JavaScript section
3. Add a new entry following the same format
4. Save the file

## Sharing on GitHub

To publish this on GitHub:

1. Create a new repository
2. Upload all three files
3. Enable GitHub Pages in repository settings
4. The HTML file will be accessible at: `https://yourusername.github.io/repository-name/artist_residencies_grants.html`
